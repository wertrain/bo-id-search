このサイトについて

2ちゃんねるの「機動戦士ガンダムバトルオペレーション晒しスレ」で言及された ID を集計し、検索しやすくしたサイトです。
あくまで言及されている ID の集計であり、プラス方向の言及であるのか、マイナス方向の晒しであるのかは区別していません。

現行運用版（バトオペ晒されID検索）との違いについて

このサイトは、ログの収集解析を自動化できる環境でテスト的に運用しているものです。
（現行運用版は、レンタルサーバの制限で自動実行の処理が行えないため、ログの収集が半自動になっています。）

既に運用されてる バトオペ晒されID検索 と機能面での違いはありません。
現行運用版のレンタルサーバの制限・運営の終了などがない限り、現行運用版の更新停止はしないつもりです。

ID の集計方法について

2ちゃんねるのネットウォッチ板・ロボットゲー板より「ガンダムバトルオペレーション晒し」と名のついたスレッドを取得し、レスで言及された PSN ID を集計しています。

レス内で PSN ID だと判断しているのは下記の条件に当てはまる文字列です。

  • アルファベットからはじまり、アルファベット, 数字, ハイフン, アンダースコア で構成された4文字(※)から16文字の文字列

PSN ID でない文字列の対処のため、以下の条件で除外を行っています。

  • 頻出文字列（Twitter, Skype, Youtube など）に関しては、ブラックリスト方式で除外
  • ID: ***** という形式で記載されたものは除外（2ch のレスがそのままコピペされている場合の対処）
  • http, ttp からはじまる URL らしいものは除外
※ PSN IDの最小文字数は3文字ですが、3文字で判定を行うとノイズが多くなってしまうので、最小4文字としています。
